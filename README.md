# Login Page

## 📌 Project Description
A simple and responsive Login Page built using HTML, CSS, and JavaScript. This project demonstrates a modern login interface with basic client-side functionality.

---

## 🚀 Features

- Responsive Login Form
- Modern UI Design
- User-Friendly Interface
- Input Validation
- Clean Layout
- Mobile Responsive
- Easy to Customize

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## 📁 Project Structure

```
Project/
│── login.html
│── login.css
│── index.js
│── README.md
```

---

## 📄 File Description

### login.html
Contains the structure of the login page.

### login.css
Contains all styling, layout, animations, and responsive design.

### index.js
Handles JavaScript functionality such as form validation and user interactions.

---

## 🎯 Project Objectives

- Practice HTML, CSS, and JavaScript
- Build a responsive login interface
- Understand form validation
- Improve frontend development skills

---

## 💻 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `login.html` in your browser.
